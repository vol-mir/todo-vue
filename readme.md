## About Project

This project To-Do с Laravel 5.8 и Vue.

## Stack

- IDE - JetBrains
- Laragon Wamp 4.0.4
- Laravel 5.8
- Bootstrap 4

## Quick Start
```shell
$ git clone https://github.com/vol-mir/todo-vue.git
$ cd landing
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate:fresh --seed
$ npm install
```
